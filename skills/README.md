# 本地 Skills

这个目录存放当前项目使用的本地技能文件，按前端和后端分组整理。

## 前端

- `frontend/vue3-component-patterns`
- `frontend/vite-project-conventions`
- `frontend/frontend-style-guide`

## 后端

- `backend/spring-boot-api`
- `backend/mysql-best-practices`
- `backend/redis-caching-patterns`

这些 skill 是围绕当前工作区技术栈编写的：

- 前端：`Vue 3 + Vite`
- 后端：`Spring Boot + MySQL + Redis`

当任务和某个 skill 的职责范围明显匹配时，优先使用对应的 `SKILL.md` 作为执行和评审依据。
